# **App Name**: BancaTrack

## Core Features:

- Subject Listing: Display a list of subjects for the user to study for their banking test.
- Video Integration: Embed YouTube videos for each subject.
- Automatic Progress Tracking: Automatically track the percentage of each video watched using YouTube IFrame API.
- Progress Saving: Save the video progress to Firebase Firestore, updating per user.
- Progress Persistence: Maintain user progress across sessions using Firebase authentication.
- Progress Visualization: Show progress bars for each video and overall subject completion.
- Overall Progress: Display overall study progress and subjects in progress.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calm, focused study environment.
- Background color: Very light blue (#F0F8FF), almost white, creating a clean backdrop.
- Accent color: Light green (#90EE90) for highlighting completed items and progress indicators.
- Body and headline font: 'Inter' for a modern and readable study environment.
- Use a left sidebar for subject navigation, with a progress summary on the home page.
- Simple, flat icons for subjects and navigation elements.
- Subtle animations for progress updates and transitions.
